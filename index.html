Welcome to Worth Data's 7802 WiFi Terminal Demonstration and Testing Server

This Server provides
1. A Connection Host for your applications using our ActiveX control running on your computer
2. A demonstration "Cloud" server that you can use to test your installation or as a template for building your own Server
3. This page that gives you access to downloading data collected using the demo Cloud Server

See the "RF 7802 Manual" for more information on using our Connection Host (or installing your own); the ActiveX control; and  the Cloud Server (or installing your own).

Cloud Server Data
To use the Cloud Server, you must first configure your Terminal to have Internet access. Consult your manual to see how to do this.

Once you have set up your Terminal, just sign-in with the default "Connection Host" settings and you will be using our Cloud Server.
Keep in mind that data is stored and accessed off the Server based on the "username", "server ID", "password" combination you have set in the Terminal.
If you want privacy, and to not see other users' demo data, change your username/serverID/password combination in yout Terminal and enter matching keys here to access your data.
